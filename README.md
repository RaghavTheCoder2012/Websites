#### Basic Structure of <abbr title="HyperText Markup Language">HTML</abbr> Document

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <header>
        <!-- Your Head -->
    </header>
    <main>
        <!-- Your Content -->
    </main>
    <footer>
        <!-- Your Foot -->
    </footer>
</body>
</html>
```
